# DS_Project1_EDA_Feature_Engineering
Production-style data preprocessing pipeline built for DecodeLabs Industrial Training (Batch 2026). Features smart missing-value imputation (Row Deletion/Median/KNN), IQR Winsorization for outliers, 5 engineered features, One-Hot Encoding, collinearity eradication &amp; Pandera schema validation. Stack: Python, Pandas, NumPy, Scikit-learn.
